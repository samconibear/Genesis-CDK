# Genisis-CDK 🚀

**Genisis-CDK** is an open‑source framework for people who want a website on AWS — **without needing to understand the underlying infrastructure, cloud architecture, or AWS CDK**.

Genisis‑CDK takes you from **nothing → a deployed website** with a few lines of code, all using AWS best practices.

---

## Genisis‑CDK is intentionally designed for developers that:
- Just want a website that works
- Have little or no AWS or IaC experience
- Dont want to manage the underlying infrastrucutre
- Want a site thats secure by default and production‑ready.

---

## 🏁 Prerequisites
You only need:
- An AWS account
- A registered domain
